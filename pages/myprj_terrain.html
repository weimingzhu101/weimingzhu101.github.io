Dynamic Terrain Generation

This is a dynamic Terrain generation project with OpenGL.
In this project, the terrain is generated from Perlin Noise function, which set in vertext shader. The normal or each triangle also be calculated dynamically in the shader. Since I don't like the color blending ways between those triangles, I made it low poly style.
This is my first try of procedural modeling.

Thermal path tracer in CUDA

This is a project for my Summer research internship. The thermal path tracer could simulate the thermal emission of objects, which works for my employer's research. I also keep working on a visible light version. This path tracer supports obj mesh, texture and 3 types of reflection: diffuse, specular and refract.
I build the scene in blender and export the scene a one obj file. In the CUDA progrom, I use imgui to match texture and set some parameters.
